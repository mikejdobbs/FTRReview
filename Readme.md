Features
--------
Currently implemented:
* Load PDF
* Display a sidebar with concerns automatically identified
* PDF thumbnail control
* Zooming
* Search
* Printing
* Forms
* Rotate page view


[1]: http://www.wxwidgets.org
[2]: https://pdfium.googlesource.com/pdfium/
